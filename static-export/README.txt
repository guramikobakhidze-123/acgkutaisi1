ACG — Audit & Consulting Group Website
=======================================
Static HTML/CSS/JavaScript Version

HOW TO OPEN IN VS CODE (Windows):
----------------------------------
1. Extract this zip to a folder (e.g. C:\acg-website)
2. Open VS Code
3. File > Open Folder > select the folder
4. Right-click on index.html > "Open with Live Server"
   (Install "Live Server" extension from VS Code Extensions panel if not installed)

OR: Simply double-click index.html to open in your browser directly.

FILES:
------
index.html        — Main/Home page
about.html        — About Us page
team.html         — Team overview page
team-murman.html  — Murman Kobakhidze profile
team-teimuraz.html — Teimuraz Gvetadze profile
team-gurami.html  — Gurami Kobakhidze profile
services.html     — All services page
contact.html      — Contact page with form

css/style.css         — All styles
js/translations.js    — Georgian/English/Russian/Turkish translations
js/main.js            — Language switcher and interactions
images/               — Team member photos

FEATURES:
---------
- 4 languages: Georgian (KA), English (EN), Russian (RU), Turkish (TR)
- Language preference saved in browser (remembers your choice)
- Fully responsive (works on mobile)
- Contact form (client-side only — shows success message)
- All team member CVs included

NOTE: The contact form is client-side only.
To make it actually send emails, you would need to connect it to a
service like EmailJS (https://www.emailjs.com) or Formspree (https://formspree.io).
